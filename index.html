AI提效带来的质变

Before

慢速原型

  • 依赖设计师排期，等待数天
  • 修改成本高，不敢大改
  • PRD 写完才发现问题
  • 容易忽略核心体验流程
  • 验证一个假设要一周以上
After

快速迭代

  • 几小时内出可交互原型
  • UX 用 WorkBuddy 做了软硬件展示网站
  • 我自己做了相机 + 打印机联动 demo
  • 先看到「产品可以怎么用」，再写 PRD
  • 拿原型让团队/用户验证

用原型驱动产品每一个环节

硬件 + 软件展示
